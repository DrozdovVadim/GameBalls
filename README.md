Для того, чтобы запустить программу необходимо зайти в vs studio code, перейти в
 file->open new folder->courseWork->gameBalls. После открытия открыть 2 новых терминала, в первом терминале должен быть путь .../CourseWork/gameBalls после чего прописать "npm start", если в терминале появилась надпись "5454", значит запустился сервер. во втором терминале открыть путь .../courseWork/gameBalls/client после чего ввести "npm start". через некоторое время откроется браузер с программой
